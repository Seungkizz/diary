# diary jsp project

## 프로젝트 계기

+ Java/Jsp를 숙지하기 위한 회원만 사용 할수있는 개인 다이어리 프로젝트
+ Servlet 을 활용한 게시판 CRUD

## 사용 기술

+ Java
+ Jsp
+ MySQL
+ JSTL
